Para utilizar o programa JurisfAI, você deve inserir os dois arquivos na extensão .docx que deseja comparar. Clique no botão superior e carregue o arquivo original. Este arquivo será levado como base para executar a comparação. O segundo botão será usado para carregar o arquivo, o qual deseja-se realizar a comparação. O programa levará alguns segundos para carregar os arquivos enviados e executar a comparação.

Será então exibida uma tabela demonstrando os parágrafos de cada arquivo onde não ocorre a correspondência. A equipe vem trabalhando para melhorar a experiência de uso do usuário, trazendo as não correspondências de forma mais simples e de modo a deixar visualmente mais fácil.
