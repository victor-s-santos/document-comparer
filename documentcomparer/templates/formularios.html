Para utilizar o programa JurisfAI, você deve inserir apenas o arquivo na extensão .docx que deseja comparar. Clique no botão superior e carregue o arquivo secundário. Este arquivo será levado como base para executar a comparação. O arquivo principal já está carregado no sistema, o qual deseja-se realizar a comparação. O programa levará alguns segundos para carregar os arquivos enviados e executar a comparação.

Será então exibida uma tabela demonstrando os parágrafos de cada arquivo onde não ocorre a correspondência. O botão comparar pode ser usado para melhorar a experiência de uso do usuário, trazendo as não correspondências de forma mais simples e de modo a deixar visualmente mais fácil.
